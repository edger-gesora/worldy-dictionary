# Wordly Dictionary SPA
A Single Page Application that allows users to search for word definitions using the Free Dictionary API.

## How to Run
1. Open this folder in VS Code.
2. Use the "Live Server" extension to open `index.html`.
3. Type a word in the search bar and press Enter.

## Features
- Dynamic API fetching using `fetch`.
- Error handling for invalid words.
- Responsive design.
- Semantic HTML and accessibility support.
